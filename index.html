Tentang Saya

Fera Andriyani adalah mahasiswa semester dua di Universitas Siber Asia yang memiliki minat besar dalam dunia menulis. Saya menggunakan menulis sebagai sarana untuk mengekspresikan pikiran dan perasaan secara jujur, sering kali melalui refleksi pribadi di buku harian. Kebiasaan ini membantu saya berpikir lebih jernih dan memahami sudut pandang secara mendalam.

Meskipun saya belum mempublikasikan karya secara luas, pengalaman ini telah melatih saya untuk merangkai kata dengan peka dan penuh makna. Saya percaya bahwa menulis adalah keterampilan penting untuk komunikasi personal dan profesional, dan saya berkomitmen untuk terus mengembangkannya sebagai bagian dari pertumbuhan saya sebagai penulis.

"Kata-kata adalah tempat perlindunganku, cara untuk merangkai emosi menjadi cerita."
